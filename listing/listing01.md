Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]

Срез имеет синтаксис:
- s[i:j] с i-ого индекса по j-1 включительно
- s[i:] c i-ого индекса до конца
- s[:i] от начала до i исключая i

```
